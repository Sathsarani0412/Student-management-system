# Student Management System

## Overview

Student Management System is a web-based application developed to simplify student administration, academic record management, attendance tracking, result management, and communication between administrators and students. The system provides separate dashboards for administrators and students, enabling efficient management of academic information through a centralized platform.

## Features

### Admin Dashboard

Administrators can:

* Add, edit, and delete student records
* Manage courses and subjects
* Record student attendance
* Add semester examination results
* Create and publish notices
* Monitor attendance statistics
* View overall system information

### Student Management

* Add new students
* Update student information
* Delete student records
* Search and filter student details


### Course Management

* Add courses
* Update course information
* Assign students to courses


### Attendance Management

Administrators can:

* Mark student attendance
* Record attendance by course
* Track present and absent students
* Monitor attendance rates

Students can:

* View personal attendance records
* Check attendance percentages
* Review attendance history

### Result Management

Administrators can:

* Add semester examination results
* Update student academic records
* Manage subject grades and performance

Students can:

* View semester results
* Monitor academic performance
* Access examination records

### Notice Management

Administrators can:

* Create notices
* Publish announcements
* Update important academic information

Students can:

* View published notices
* Access academic announcements
* Stay informed about university activities and updates



### User-Friendly Interface


* Easy navigation
* Interactive dashboards
* Modern user interface

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite

## Database Functionality

The SQLite database stores:

* Student Information
* Course Information
* Attendance Records
* Examination Results
* Notices and Announcements
* Academic Data

## System Architecture

Administrator
↓
Web Application
↓
Node.js Server
↓
SQLite Database

Student
↓
Web Application
↓
Attendance, Results, and Notices

## Key Functionalities

### Student Record Management

Efficiently manage student information and academic records.

### Attendance Tracking

Record and monitor attendance for individual students and courses.

### Result Management

Store and manage semester examination results and academic performance data.

### Notice Management

Publish and distribute important notices and announcements.

### Reporting and Statistics

Generate attendance summaries, academic insights, and dashboard analytics.
